Тема, мета, місце розташування лаби

Тема: Типи та способи верстки

Мета: придбати практичні навички роботи верстки сторінок засобами CSS, верстки на основі плаваючих елементів, з’ясувати переваги та недоліки типів макетів веб-сторінок, придбати практичні навички роботи верстки сторінок засобами CSS та FLEXBOX

Місце розташування лаби: Github Pages

Вихідний код проекту: Github

Тип верстки у проекті: Блокова верстка (Block Layout)

Плюси:

  • Простота у використанні та розумінні: Блокова верстка дуже інтуїтивна і легко зрозуміла, особливо для початківців. Елементи розташовуються один під одним у вигляді блоків.
  • Гнучкість: Блоки можна легко переміщувати, змінювати їхні розміри та стилі. Це забезпечує хорошу гнучкість у дизайні.
  • Сумісність з усіма браузерами: Блокова верстка підтримується всіма сучасними браузерами, що забезпечує сумісність і стабільність.
  • Простота візуалізації: Завдяки зрозумілій структурі блоки легко візуалізувати та налаштовувати.

Мінуси:

  • Неоптимальна для складних макетів: Блокова верстка може стати громіздкою при створенні складних і багатошарових макетів. Для складних макетів можуть бути більш підходящі інші методи, такі як Flexbox або CSS Grid.
  • Менша ефективність при адаптивному дизайні: Хоча блокова верстка може використовувати медіа-запити для адаптивного дизайну, вона менш ефективна в порівнянні з сучасними методами верстки, такими як Flexbox і Grid.
  • Обмежені можливості вирівнювання: Блокова верстка має обмежені можливості вирівнювання елементів по горизонталі і вертикалі. Це може ускладнити створення складних макетів з точним вирівнюванням елементів.
