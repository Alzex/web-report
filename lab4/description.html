Створення десктопної, планшетної та мобільної версії

Для створення адаптивної верстки використовуються медіа-запити, які дозволяють застосовувати різні стилі в залежності від розміру екрану пристрою. Ось основні кроки для створення верстки для різних пристроїв:

Десктопна версія

Десктопна версія зазвичай створюється першою, оскільки на великих екранах найзручніше розташовувати елементи інтерфейсу та контент. Наприклад, для десктопної версії можна використовувати широку сітку з кількома колонками, великими зображеннями та багатим текстовим контентом.

Планшетна версія

Для планшетів зазвичай використовуються медіа-запити для пристроїв з шириною екрану до 1200 пікселів. В цій версії інтерфейс має бути більш компактним, можливо, зменшення кількості колонок у сітці та зменшення розміру зображень. Наприклад:


                @media (max-width: 1200px) {
                    .container {
                        padding: 10px;
                    }
                    .column {
                        width: 100%;
                    }
                }

Мобільна версія

Для мобільних пристроїв зазвичай використовуються медіа-запити для пристроїв з шириною екрану до 768 пікселів. Інтерфейс має бути максимально спрощеним, з великими кнопками та текстом, щоб забезпечити зручність використання на маленьких екранах. Наприклад:


                @media (max-width: 768px) {
                    .container {
                        padding: 5px;
                    }
                    .column {
                        width: 100%;
                        margin-bottom: 20px;
                    }
                }
