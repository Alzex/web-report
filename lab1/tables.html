Способи створення таблиць:

  • HTML-таблиці:
    • Створюються за допомогою тегів table, tr, th, td.
    • Мають обмежені можливості для оформлення.
    • Використовуються для відображення даних у вигляді таблиць.
  • CSS-таблиці:
    • Створюються за допомогою CSS-властивостей display, grid, flex.
    • Мають широкі можливості для оформлення.
    • Використовуються для створення складних макетів.

Теги та призначення:

Тег Призначення
table Визначає таблицю
tr Визначає рядок таблиці
th Визначає заголовок стовпця або рядка таблиці
td Визначає комірку таблиці
