Обробник подій

Обробники подій (event handlers) дозволяють виконувати JavaScript-код у відповідь на події, такі як натискання кнопок, рух миші тощо. Наприклад, ви можете використовувати атрибути HTML, такі як onclick, onmouseover тощо:

<button onclick="alert('Button clicked!');">Натисніть мене</button>

Або ви можете додавати обробники подій за допомогою наступного JavaScript-коду:

document.getElementById('myButton').addEventListener('click', function() { alert('Button clicked!'); } );
